# Пайплайн работы приложения:
Пользователь спрашивает бота в вк -> бот берет сообщение и кидает его в бэкенд -> бэкенд кидает сообщение в нейронку обученную -> нейронка понимает контекст сообщения и выдает обратно в бэкенд теги для вывода ответа и сопроводительный текст -> бэкенд по тегам вытягивает информацию из базы данных, формирует ответ для бота и кадает сообщение боту -> бот кидает это всё пользователю -> пользователь получает верное сообщение и счастливый идет дальше

# Вк бот
## Язык: Python
## Требования к боту:
1) отдавать сообщения от пользователя в бэкенд.
2) принимать сообщения от бэкенда и отправлять их обратно пользователю.
## подробно
1. бот запускается
2. бот принмает сообщение от пользователя через апи
3. бот форматирует это сообщение под формат json (формат json будет в этом же документе)
4. бот получает из файла config.py в котором написаны ip сервера, данные сервера
5. бот по полученным данным сервера и сформированному json кидает пост запрос в сервер
6. бот ждет ответа от сервера в формате json (формат будет описа в этом же документе)
7. бот расформатирывает сообщение из json
8. бот кидает расформированное сообщение пользователю
9. бот ждет нового сообщение от пользователя и идем на пункт 2.

# бэкенд
## язык: TypeScript
## Требования к бэкенду:
1)	принимать сообщение от вк бота и отдавать новые сообещния обратно
2)	отдавать запрос в нейронку и получать ответ от нейронки
3)	должен уметь отдавать запрос на получение данных из бд и получать его
4)	Формировать конечное сообщение, основанное на тексте полученного от нейронки и информации полученной от запроса из бд
